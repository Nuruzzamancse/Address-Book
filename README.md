# Address-Book

Online Contact Management System in PHP & MySql

To use the system users have to register using his/her username and password

After logging in user can add, edit or delete address book entry

Each address entry in address book contain commonly used fields (full name, nickname, photos, multiple phone no., address, website, date of birth etc.).

Address book should be searchable based on address fields

The Application has option to download all contact information in CSV format
